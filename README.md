##  Development

Install dependencies:

```bash
$ npm i || npm install
```

Now, you can start a local web server by running:

```bash
$ npm start
```

and then you can open http://localhost:3000 to view it in the browser.

## Basic Features

- [x] Create
- [x] Read
- [x] Update
- [x] Delete
